## Wonderland Clojure Kata: Alphabet Cipher

Looking for material to learn Clojure I stumbled across an interesting Github repo:
[gigasquid/wonderland-clojure-katas](https://github.com/gigasquid/wonderland-clojure-katas)
and I decided that solving the katas described in the repo would be my first project
to learn Clojure.

[alphabet-cipher](https://github.com/gigasquid/wonderland-clojure-katas/tree/master/alphabet-cipher)
is the first kata in the list.

<!--truncate-->

Given that I have been using Scala in my day work, I started solving the kata
using Scala and only then I solved the kata using Clojure.
